Graphics and CSS Pseudo Elements

In this project, I have styled graphics using CSS Pseudo class elements. There are two graphics: the first being the flag type green shape used in the header and the second being the arrows used in the unordered list below.
The source code for this project can be easily seen using the view source options in your browser. Feel free to use this in any project you may want to.
