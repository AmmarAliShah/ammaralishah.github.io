Ello World...again!

03 July 2015

Again? Yes again! I had a blog running on this domain before but I lost it - more on that here. So, as promised in that post, I'm back but this time, with improvements:

What's new?

A lot actually. First, my previous blog was running on WordPress and was hosted for free at 000WebHost. Now the price for free was that whenever someone accessed my blog or reloaded it, an ad popped up which was very annoying. Now, my blog is running on Gost and is hosted for free, on GitHub Pages, but this time, no ads. Now here's the catch, GitHub Pages only supports static sites so WordPress and other PHP or node.js based CMS(liked Ghost) are not supported. Wait! You said Ghost isn't supported, but your blog is running on Ghost and hosted on GitHub pages. Ah yes! Ghost isn't natively supported but there's a way using which you can run Ghost, read about it here.

What else?

A few other things too:

  • I started Harvard - CS50x on edx last week. It's 12 week long, 13 if you count week 0, but I plan on ending it before the start of August. Let's see how that goes!

  • I signed up to Learn Basic Mandarin on edx. The class starts from July 6th.

  • I also started reading Eloquent JavaScript a few days ago. It's an amazing book and I'm learning a lot! I'm keeping track of my solutions to the challenges in a repository on GitHub - you can check it out over here

And?

And that's it! I'm still working my way through Free Code Camp and I hope I start working on those non-profit projects soon :-)
I'll be writing more often now and I'll soon set a regular updating routine after I start college in August. That's all for now guys!
Stay happy and stay blessed!
